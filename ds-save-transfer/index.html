    This tutorial will guide you through moving a DS cart's save to TwiLightMenu++. There are two ways we can go about moving the save, one way is through the save manager Checkpoint, the other way is through Godmode9.

    What you need:

    • A 3DS with Luma CFW
    • A legally dumped .nds file
    • Any file manager of your choice, or Checkpoint

    Checkpoint instructions

    1. On your 3DS, open Checkpoint and wait for the cartridge to load.
    2. Select the DS Cartridge press A then L to back up the save.
    3. Close Checkpoint and turn off your 3DS.
    4. Eject the SD Card and insert it into your computer (or any other device that can write to an SD Card).
    5. Look for the folder which contains your .nds backups.
    6. Inside this folder, make another folder named saves.
    7. On your computer, navigate to SD > 3ds > Checkpoint > saves > (your DS cart's name) > (backup name) and copy the backed up save to the saves folder in the location where your .nds backups are.
    8. Rename the save file so it matches the name of the .nds file(for example if the .nds file is called “game.nds” the save file must be called “game.sav”
    9. Test out your new save in TWiLightMenu++.

    GM9 instructions

    1. Open gm9 by holding start while pressing the power button.
    2. Navigate to the [C:] GAMECART drive and press A to open it, then select the .sav file and press Y to copy it to the clipboard. (GM9 takes a while to open the gamecart drive, if you find yourself waiting for the contents to load for up to a minute, check if your cart can be read by the console normally.)
    3. Create a folder in the location of your dump (press R + Y in the directory to make a folder) and name it saves, open it then press Y to paste the .sav file from the [C:] GAMECART drive.
    4. Select the save file, press R + X and rename it so it matches the name of the dump.
    5. Test out your save file in TWiLightMenu++.
